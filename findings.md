# Findings

## Observations
* Hash-ID was used to identify the supplied hash type.
* Hashcat was configured with the appropriate attack mode.
* Dictionary attacks using the RockYou wordlist successfully demonstrated the risk of weak passwords.
* Password complexity directly impacts resistance to password recovery attempts.

## Security Recommendations
* Use strong and unique passwords.
* Avoid dictionary words and predictable patterns.
* Enable Multi-Factor Authentication (MFA).
* Use modern password hashing algorithms with salting.

## Key Learning
* Hash identification
* Password auditing
* Dictionary attack methodology
* Password security best practices
